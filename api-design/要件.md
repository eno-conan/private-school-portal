# イベント運営システム

### 大項目
1. 会員
2. イベント

### 機能
1. 会員
   1. 会員登録
   2. 会員情報表示
   3. 会員情報更新
   4. 退会
2. グループ（コミュニティ）
   1. グループ作成
   2. メンバー追加・削除
3. イベント
   1. イベント作成
      1. イベント名
      2. 開催日時
      3. 場所
      4. 参加可能人数
      5. タグ付け
   2. イベント申込機能
   3. イベント開催
   4. イベント後感想？
